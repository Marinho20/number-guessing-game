# Guess Numbers Game

An interactive CLI number guessing game built in Python. Features multiple difficulty levels, hints, high scores, and a replay system. Designed to practice Python fundamentals, file handling, and software design.

## Features

- Multiple difficulty levels: Easy, Medium, Hard, and Custom
- Real-time hints (higher/lower)
- High score tracking with persistent storage
- Replay system for multiple rounds
- Modular code structure with functions grouped into regions for better readability
- Input validation for robust user experience
- Interactive menu system for easy navigation

## How to Run

1. Download the repository as a ZIP or clone it.  
2. Extract the files if needed.  
3. Open a terminal or command prompt and navigate to the project folder.  
4. Run the game: "python main.py"

Notes:
- Requires Python 3.10+
- No extra packages needed
